# Awesome Nano Banana Pro Gallery 🍌

## 🌟 프로젝트의 의미
이 프로젝트는 **Nano Banana Pro** 모델의 잠재력을 최대한 끌어내기 위해 엄선된 프롬프트를 시각적으로 탐색하고 배울 수 있는 **오픈 소스 프롬프트 갤러리**입니다. 

단순한 텍스트 나열이 아닌, 실제 생성 결과물과 함께 프롬프트의 구조와 미학을 직접 확인하고 복제하여 자신의 창의적 작업에 즉시 적용할 수 있도록 설계되었습니다. 특히 인쇄 및 PDF 저장 기능을 통해 고품질의 프롬프트 가이드를 오프라인에서도 소장하고 활용할 수 있는 가치를 제공합니다.

## 🚀 주요 기능 및 특징
- **엄선된 큐레이션**: 사진 리얼리즘, 창의적 실험, 교육 및 지식 시각화 등 최적화된 프롬프트 제공
- **지능형 프린트 레이아웃**: 라이트/다크 테마를 유지하며 페이지당 2장씩 정갈하게 출력되는 PDF 최적화 레이아웃
- **다국어 지원 (Ko/En)**: 한국어와 영어를 자유롭게 전환하며 프롬프트 학습 가능
- **모던한 웹 디자인**: Framer Motion 기반의 부드러운 애니메이션과 반응형 UI
