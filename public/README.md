## 📖 프로젝트 소개

이 프로젝트는 우수한 [Awesome Nano Banana Pro](https://github.com/ZeroLu/awesome-nanobanana-pro) 프롬프트 컬렉션을 인터랙티브하고 시각적으로 멋진 웹 갤러리로 변환합니다. 

마크다운 파일을 탐색하는 대신, 사용자는 다음과 같은 기능을 갖춘 현대적이고 반응형 인터페이스를 통해 프롬프트를 탐색할 수 있습니다:

- 🎨 **아름다운 카드 레이아웃** - 샘플 이미지가 포함된 직관적인 갤러리
- 🌓 **라이트/다크 모드** - 취향에 맞는 테마 선택
- 🌍 **이중 언어 지원** - 한국어/영어 자유롭게 전환
- 🖨️ **PDF 내보내기** - 페이지당 2장씩 깔끔하게 인쇄
- 🔍 **카테고리 필터** - 원하는 스타일만 골라보기
- 📋 **원클릭 복사** - 프롬프트를 클릭 한 번으로 복사
- ✨ **부드러운 애니메이션** - 세련된 사용자 경험

---

## 🎯 카테고리별 프롬프트

갤러리는 10개의 신중하게 선별된 카테고리로 프롬프트를 구성합니다:

### 1. 📸 사진 리얼리즘 (Photo Realism)
전문 사진 스타일 및 기법을 활용한 프롬프트

**주요 프롬프트:**
- 시네마틱 인물 사진
- 심도가 있는 제품 사진
- 거리 사진 미학
- 건축 사진

### 2. 🎨 창의적 실험 (Creative Experiments)
구성과 창의성의 경계를 넓히는 실험적 프롬프트

**주요 프롬프트:**
- 아이소메트릭 미니어처 세계
- 이중 노출 효과
- 초현실적 구성
- 추상 예술 스타일

### 3. 📚 교육 및 지식 (Education & Knowledge)
개념을 명확한 교육용 일러스트로 변환

**주요 프롬프트:**
- 개념 시각화 (텍스트를 인포그래픽으로)
- 시각적 레시피 가이드
- 3D 미니어처 디오라마
- 재무 데이터 시각화

### 4. 🛍️ 전자상거래 및 가상 스튜디오
전문 제품 사진 솔루션

**주요 프롬프트:**
- 가상 모델 착용
- 전문 제품 사진
- 라이프스타일 제품 샷
- 360° 제품 뷰

### 5. 💼 업무 및 생산성
스케치를 전문 프레젠테이션으로 변환

**주요 프롬프트:**
- 스케치를 프레젠테이션으로
- 다이어그램 향상
- 와이어프레임을 디자인으로
- 데이터 시각화

### 6. 🖼️ 사진 편집 및 복원
스마트 이미지 편집 및 복원 기법

**주요 프롬프트:**
- 사진 복원
- 배경 제거
- 색상 보정
- 이미지 향상

### 7. 🏠 인테리어 디자인
평면도에서 완전한 디자인 프레젠테이션 생성

**주요 프롬프트:**
- 평면도 시각화
- 공간 디자인 컨셉
- 가구 배치
- 재질 및 텍스처 렌더링

### 8. 📱 소셜 미디어 및 마케팅
매력적인 썸네일 및 홍보 자료 제작

**주요 프롬프트:**
- 유튜브 썸네일 디자인
- 인스타그램 스토리 템플릿
- 마케팅 배너
- 소셜 미디어 그래픽

### 9. 🌏 일상 생활 및 번역
메뉴, 표지판 및 디지털 콘텐츠를 원활하게 번역

**주요 프롬프트:**
- 메뉴 번역
- 표지판 번역
- 문서 번역
- 실시간 텍스트 번역

### 10. 👤 소셜 네트워킹 및 아바타
아바타, 밈 및 소셜 미디어 콘텐츠 제작

**주요 프롬프트:**
- 아바타 생성
- 밈 제작
- 프로필 사진 향상
- 캐릭터 디자인

## 🛠️ 기술 스택

### 프론트엔드 프레임워크
- **React 18.2** - 최신 React 기능 활용
- **TypeScript 5.2** - 타입 안전성 보장
- **Vite 5.0** - 초고속 빌드 도구

### 스타일링 및 UI
- **Tailwind CSS 3.3** - 유틸리티 우선 CSS 프레임워크
- **Framer Motion 10.16** - 부드러운 애니메이션
- **Lucide React** - 아름다운 아이콘 라이브러리

### 코드 표시
- **React Syntax Highlighter** - 프롬프트 구문 강조
- **Prism 테마** - 라이트/다크 모드용 코드 테마

## 🚀 시작하기

### 필수 요구사항

```bash
Node.js >= 18.0.0
npm >= 9.0.0
```

### 설치 방법

```bash
# 저장소 복제
git clone https://github.com/jvisualschool/Nano-Banana-Pro-Gallery.git

# 프로젝트 디렉토리로 이동
cd Nano-Banana-Pro-Gallery

# 의존성 설치
npm install

# 개발 서버 시작
npm run dev
```

애플리케이션은 `http://localhost:5173`에서 사용할 수 있습니다.

### 프로덕션 빌드

```bash
# 최적화된 프로덕션 빌드 생성
npm run build

# 프로덕션 빌드 미리보기
npm run preview
```

## ✨ 주요 기능

### 🎨 아름다운 UI/UX
- 세부 사항에 주의를 기울인 현대적이고 깔끔한 디자인

### 🌓 테마 지원
- 라이트 및 다크 모드
- 자동 테마 저장
- 부드러운 테마 전환
- 테마 색상을 유지하는 인쇄 친화적 레이아웃

### 🌍 다국어 지원
- 완전한 한국어 및 영어 지원
- 쉬운 언어 전환
- 번역된 프롬프트 및 UI 요소

### 🖨️ 고급 인쇄 기능
- 인쇄를 위한 최적화된 2열 레이아웃

- 인쇄 시 전체 프롬프트 텍스트 표시
- 출처 표시 포함
- 테마 인식 인쇄 (라이트/다크 색상 유지)

### 📋 개발자 친화적
- 원클릭 프롬프트 복사
- 코드 프롬프트용 구문 강조
- JSON 및 텍스트 프롬프트 지원
- 외부 소스 링크
