# Screenshots

이 폴더에는 Nano Banana Pro Gallery의 스크린샷이 포함됩니다.

## 권장 스크린샷 목록

1. **light-mode-gallery.png** - 라이트 모드 메인 갤러리 뷰
2. **dark-mode-gallery.png** - 다크 모드 메인 갤러리 뷰
3. **prompt-modal.png** - 프롬프트 상세 모달
4. **splash-modal.png** - 스플래시/정보 모달
5. **print-layout.png** - PDF 인쇄 레이아웃 (2열)
6. **mobile-view.png** - 모바일 반응형 뷰

## 스크린샷 가이드라인

- 해상도: 최소 1920x1080 권장
- 형식: PNG 또는 JPG
- 브라우저: Chrome 또는 Safari 최신 버전
- 실제 콘텐츠가 잘 보이도록 캡처
